# Asheville Airbnb Market Analysis

SQL and Power BI project analyzing Airbnb listings in Asheville, North Carolina.

## Tools
- SQL
- Power BI

## Key Findings
- Average nightly price: $181
- Median nightly price: $128
- Entire home/apartment listings dominate the market.
- Review activity is concentrated among a small number of hosts.
- Asheville shows strong demand and potential for growth.
